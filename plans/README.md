# ExecPlans

This directory contains task-specific living implementation plans.

Use an ExecPlan for complex or multi-session work. Each plan must be self-contained and kept current as implementation proceeds.

ExecPlans are working documents, not permanent architecture references. Durable architectural or product decisions discovered while executing a plan must also be appended to `DECISIONS.md`.

The ExecPlan format is defined in `agents/execplan.md`.

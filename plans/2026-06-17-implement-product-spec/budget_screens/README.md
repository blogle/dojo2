# dojo budget page screen mockups

This bundle contains the curated, spec-aligned budget page mockup screens generated for implementation reference.

## Contents

1. `01-budget-default.png` — Default Budget page with compact shell, metric strip, and hierarchical category table.
2. `02-budget-historical-readonly.png` — Budget page in read-only historical/as-of mode.
3. `03-add-category-group-modal.png` — Add category group flow with Add dropdown and compact form modal.
4. `04-add-category-modal.png` — Add category flow with parent group and goal editor.
5. `05-reordering-mode.png` — Dedicated reordering mode with pending changes and drag handles only in reorder state.
6. `06-category-detail-fund-dropdown.png` — Category detail modal with spec-correct Fund dropdown and custom amount row.
7. `07-funding-shortcuts-negative-preview.png` — Funding flow preview showing resulting category balance and negative Available to budget warning.
8. `08-move-funds-modal.png` — Move funds flow with category-to-category source/destination and resulting balances.
9. `09-fund-category-group-modal.png` — Fund category group flow using Available to budget and partial funding summary.
10. `10-retired-categories-modal.png` — Retired categories modal with Restore actions.
